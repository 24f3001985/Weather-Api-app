# Weather App

A simple weather application built with HTML, CSS, and JavaScript.

## Features

- Search weather by city
- Current temperature
- Weather condition
- Humidity
- Wind speed
- Weather icon

## Setup

1. Get a free API key from WeatherAPI.
2. Open the app.
3. Enter your API key.
4. Enter a city name.
5. Click Get Weather.

## Technologies

- HTML
- CSS
- JavaScript
- WeatherAPI
